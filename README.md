# Student-Performance-Analytics-Dashboard-
The Student Performance Analytics Dashboard is a data analysis project built using Python to explore academic performance patterns across a large synthetic student dataset. The project analyzes how factors such as attendance, socio-economic background, parental education, school type, and demographics influence student outcomes.
